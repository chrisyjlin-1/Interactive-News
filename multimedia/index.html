Reflection on assignment

It was pretty easy to use the Knight Lab tool to create the interactive timeline but I did run into some problems with updating the timeline when I edited the spreadsheet. It was pretty easy to use as long as I had the information and data that I needed in order to upload it to the tool. The tool fit into the story this time because it was able to visualize the timeline of Dean's career from her first single to her first Grammy. It would help engage audiences because it's quite easy to use and is able to sum up her career very easily without having to write as much. I also liked the ability to include multimedia elements like her music videos, albums and Instagram posts. One probelm that I could not solve was that Instagram posts had to be opened on Instagram in order to see the content. That is why I opted to use mostly YouTube and Spotify embeds/links.
